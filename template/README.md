## SBT project compiled with Scala 3

A basic Scala 3 project with a main class and a MUnit test.


## Compile

You can compile the code with:

```
sbt compile
```

## Run tests

You can run tests with:

```
sbt test
```


## Interactive console

You can jump into an interactive Scala 3 REPL with:

```
sbt console
```
